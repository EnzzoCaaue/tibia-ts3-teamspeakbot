## TS3 Tibia BOT

### Install

```bash
git clone https://github.com/ethkat/tibia-ts3-teamspeakbot.git
yarn install
```

### dev

```bash
yarn dev # start dev server at localhost:3000
```

### lint

```bash
yarn lint
```
